Tutorial/Walkthrough

The steps and final table are published in this repo’s GitHub Page. Important steps: 1. Creating a complex map plot - both interactive via {ggiraph} + {ggplot2} and static via only {ggplot2}. 2. Creating a function to generate AZ Indigenous Tribal subplot maps (inspired by the 2022 Posit Table Contest Submission by @jfangmeier). 3. Creating Header and Footer content for the main table. 4. Creating the main table with {reactable} and {reactablefmtr}.
